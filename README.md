# Installation
```
git clone https://github.com/edfloreshz/cosmic-tasks.git
cd cosmic-tasks
sudo just install
```

# Build
```
git clone https://github.com/edfloreshz/cosmic-tasks.git
cd cosmic-tasks
cargo build
```

## Dependencies
- [libcosmic](https://github.com/pop-os/libcosmic?tab=readme-ov-file#building)
- sqlite3

Ubuntu
```
sudo apt install libsqlite3-dev
```

Fedora
```
sudo apt install sqlite-devel
```
